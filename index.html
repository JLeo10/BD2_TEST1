Consola de Control de Eventos

Ingesta de Big Data en Tiempo Real - Proyecto Bases de Datos 2
Interacción Simple
Registra un evento básico de clic con parámetros fijos para demostrar el envío asíncrono.
{
  "event_name": "clic_ingenieria",
  "params": {
    "seccion": "botones",
    "integrante": "equipo_expo"
  }
}
Simulación de Registro
Registra un evento que simula un registro de estudiante, ideal para explicar esquemas flexibles NoSQL.
{
  "event_name": "registro_estudiante",
  "params": {
    "metodo": "demo_web",
    "seccion": 2000
  }
}
Simulación de Query
Envía parámetros numéricos y booleanos. Muestra cómo se ven estos tipos de datos en un campo RECORD de BigQuery.
{
  "event_name": "simulacion_query",
  "params": {
    "tiempo_ms": 450,
    "exito": true
  }
}
